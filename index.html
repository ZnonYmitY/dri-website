这是 DRI 网站的公开首版，用于沉淀项目介绍、核心功能、内容结构和后续迭代方向。

Overview

网站内容概要

项目定位

用一个公开网页承载 DRI 相关信息，让访问者快速理解项目背景、当前内容和未来扩展方向。

首屏信息

首屏直接呈现网站名称和当前用途，帮助访问者明确这是一个持续维护中的公开站点。

内容维护

通过项目内的 WEBSITE.md 同步记录内容概要、功能特点、交互细节和部署方式。

Features

功能特点

  • 公开访问

    站点可通过 GitHub Pages 发布，生成稳定的公开链接。

  • 静态部署

    当前首版不依赖后端服务，部署和维护成本较低。

  • 可持续迭代

    后续可以逐步加入更多页面、数据模块、下载内容或交互组件。

Interaction

交互细节

页面导航

顶部导航锚点跳转到内容概要、功能特点和交互细节。页面在桌面端和移动端都保持清晰可读。

响应式布局

首版采用静态响应式布局，内容区会根据屏幕宽度自动调整排列方式。

DRI 网站将随项目进展持续更新。
